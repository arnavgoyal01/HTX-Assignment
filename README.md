# Instruction 
1. Download packages by running the below command in the terminal
```bash
$ pip install -r requirements.txt
```
2. Open repo in VSCode code editor using following command
```bash 
$ code .
```
or start a Jupyter Lab server using the following command
```bash 
$ jupyter lab
```
And copy the localhost list provided into a browser of you choosing
3. Run Submission.ipynb normally
